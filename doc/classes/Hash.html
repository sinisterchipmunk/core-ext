rename(to)

Takes a hash whose keys must match keys in this hash. Those keys will be renamed to match the corresponding value in the specified hash.

Keys not found are ignored.

Returns self.

Example: 

  { :a => 1 }.rename(:a => :b)
    => {:b => 1}
